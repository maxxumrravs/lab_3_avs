# lab_3_avs
Лабораторная работа №3 по курсу «Архитектура вычислительных систем». Работа с Git: ветвление, слияние, разрешение конфликтов, rebase, merge. 

Порядок выполнения работы пошагово:

Часть 1
Установлен git, curl, выполнена команда из задания, получена папка lab3_repo_501473.
В файл requirements.txt добавлена строка "зачет", изменения зафиксированы в коммите. 

Часть 2
В тот же файл добавлена строка "не зачет", создан второй коммит.

Часть 3
Переключились на ветку gguf, в ней файл requirements.txt обновлён (оставлена строка "зачет"), сделан коммит.

Часть 4
Сравнены два последних коммита с помощью команды git diff, результат сохранён в файл diff.txt.

Часть 5
Выполнен откат к третьему коммиту в ветке main, внесены изменения и создан новый коммит.

Часть 6
Выполнена операция rebase ветки gguf на main. Возникший конфликт разрешён вручную — оставлено корректное содержимое файла, изменения зафиксированы

Часть 7
Переключили по очереди на ветки feature1 и feature2, в каждую были внесены изменения. Обе ветки слиты в main с разрешением конфликта при слиянии.

Часть 8
Создана ветка release от актуального состояния main, тем самым в неё включены все выполненные ранее изменения.
